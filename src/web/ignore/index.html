Configuration Guide

hydro.yml Structure

project: "my-project"
languages: ["javascript", "typescript", "python"]
scan:
  include: ["src", "lib"]
  exclude: ["node_modules", "dist", ".git"]
rules:
  complexity:
    maxFunctionLines: 50
    maxCyclomaticComplexity: 10
  naming:
    style: "camelCase"
  dependencies:
    upgradeStrategy: "safe"
outputs:
  reports: ".hydro/reports"
  codemods: ".hydro/codemods"

CI/CD Integration

GitHub Actions

name: Hydro Analysis
on: [push, pull_request]
jobs:
  hydro:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '18'
      - name: Install Hydro
        run: npm install -g hydro-cli
      - name: Run Analysis
        run: hydro analyze --all --format json > hydro-report.json
      - name: Upload Report
        uses: actions/upload-artifact@v4
        with:
          name: hydro-report
          path: hydro-report.json

Common Issues

"No files found"

Check your scan configuration

hydro scan --verbose

"Analysis timeout"

Large codebase - use filters

hydro analyze --threshold 3

"Permission denied"

Check file permissions

chmod +x node_modules/.bin/hydro

Debug Mode

Enable Debug

HYDRO_DEBUG=1 hydro analyze

Verbose Output

hydro analyze --verbose

Dry Run

hydro analyze --dry-run

Hydro Commands

Master Hydro's powerful command-line interface with detailed explanations and examples

Core Commands

hydro init Initialize

Initialize Hydro configuration in your project

# Interactive setup
hydro init
# Use template
hydro init --template advanced
hydro scan Discovery

Scan and discover files in your project

# Basic scan
hydro scan
# Full analysis
hydro scan --full
hydro analyze Analysis

Run comprehensive code analysis

# All analyzers
hydro analyze --all
# Specific analysis
hydro analyze --cycles --complexity

Advanced Commands

hydro codemap Visualization

Generate visual maps of your codebase

# Dependency map
hydro codemap --type dependencies
# Complexity heatmap
hydro codemap --type complexity
hydro write Documentation

Generate documentation and files

# Generate README
hydro write --readme
# Use template
hydro write --readme --template comprehensive
hydro update Maintenance

Update Hydro to the latest version

# Check for updates
hydro update --check
# Update to latest
hydro update
hydro ai AI Assistant

AI-powered code analysis and suggestions

# Ask questions about code
hydro ai "What does this function do?"
# Get suggestions
hydro ai --suggest
hydro plugin Plugin Management

Manage and install plugins for extended functionality

# List installed plugins
hydro plugin list
# Install plugin
hydro plugin install my-analyzer
